# Pressing Traps Patterns

## Core Idea

Invite the opponent into a specific pass, then collapse around the receiver.

A pressing trap is not random pressure. The team shapes the opponent's decision before the press is launched.

## Base Structure

```text
Open lane: bait
Near-side players: hidden pressure
Far-side players: compact cover
```

## Main Triggers

* pass into full-back near touchline
* pass into pivot facing own goal
* pass into weak-foot center-back
* backward pass after opponent pressure
* receiver takes touch toward sideline

## Player Movements

* first presser curves run to block return pass
* second presser closes forward option
* third player jumps to inside support
* full-back steps to winger
* center-back covers depth behind the trap

## Typical Pattern

```text
CB baited to RB
LW presses RB from outside
LCM jumps to DM
LB locks RW
ST blocks return to CB
```

Goal: make the opponent believe a pass is free, then remove every exit.

## Strengths

* creates predictable turnovers
* turns opponent's build-up habits against them
* can be repeated in specific zones

## Risks

* trap fails if timing is late
* one-touch combinations can escape
* weak-side switch must be protected

## Elite Usage

Used heavily by pressing teams and possession teams that want to control where the opponent is allowed to play.
