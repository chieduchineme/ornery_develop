# Cover Shadow Defending Patterns

## Core Idea

Press one player while using body shape to block the pass into another.

Cover shadow defending lets a player influence two opponents at once.

## Base Structure

```text
Presser -> ball carrier
Presser's body angle blocks hidden receiver
```

## Main Triggers

* center-back carries ball toward midfield
* goalkeeper looks for pivot
* full-back receives with inside midfielder nearby
* team wants to deny one key playmaker
* press starts from curved runs

## Player Movements

* striker curves run to block pass into defensive midfielder
* winger presses full-back while hiding central midfielder
* midfielder presses pivot while blocking attacking midfielder
* defender approaches from angle that blocks inside turn
* teammates lock the visible options

## Typical Pattern

```text
ST presses LCB
ST's run blocks DM
RW prepares for pass to LB
RCM marks LCM
```

Goal: force the ball carrier toward the team's chosen trap.

## Strengths

* reduces need to mark every player directly
* improves pressing efficiency
* controls the opponent's preferred progression lane

## Risks

* bad body angle opens the hidden pass
* clever movement can escape the shadow
* requires coordinated pressure behind the first presser

## Elite Usage

Essential in modern pressing. Elite forwards are often selected partly for their ability to press with intelligent body shape.
